# Remotion Video Generator

Render videos from `video-manifest.json` timeline. **Timeline is already built by CLI - just use it.**

| Composition | Dimensions | Captions | Use Case |
|-------------|------------|----------|----------|
| `YouTubeVideo` | 1920x1080 (16:9) | No | YouTube, Vimeo |
| `TikTokVideo` | 1080x1920 (9:16) | Yes (word-by-word) | TikTok, Reels, Shorts |

## Quick Start

```bash
pnpm install
pnpm remotion   # Preview in Remotion Studio
```

## Render

**The `video-manifest.json` already contains the complete timeline. No need to build or transform anything - just pass it directly.**

**YouTube (landscape, no captions):**
```bash
pnpm exec remotion render YouTubeVideo out/youtube.mp4 \
  --props='{"timeline":'$(cat public/video-manifest.json | jq -c .timeline)',"showCaptions":false}'
```

**TikTok (vertical, with captions):**
```bash
pnpm exec remotion render TikTokVideo out/tiktok.mp4 \
  --props='{"timeline":'$(cat public/video-manifest.json | jq -c .timeline)',"showCaptions":true}'
```

## How It Works

1. CLI generates `video-manifest.json` with complete `timeline` (audio, images/videos, captions with timestamps)
2. Pass `timeline` directly to composition - **no transformation needed**
3. Render with desired composition (YouTube or TikTok)

Same timeline, different output formats.

```
├── video-manifest.json         # Generated by CLI (timeline ready to use)

```

## video-manifest.json

Generated by CLI. **Timeline is ready - use as-is:**

```json
{
  "timeline": {
    "elements": [...],    // Images/videos with timing + animations
    "audio": [...],       // Audio tracks with timing
    "text": [...]         // Captions with character-level timestamps
  },
  "totalDurationInFrames": 135,
  "fps": 30
}
```
